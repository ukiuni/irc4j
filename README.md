irc4j
=====
irc4j is IRC client/server implementation for java.

# Install

## server
download archive from [here](https://github.com/ukiuni/irc4j/releases) and unzip, exec one script. just it.

```shell:
unzip aris.x.x.x.zip
cd bin
chmod +x ./airs.sh
nohup ./airs.sh 2>&1 >> irc.log &
```
